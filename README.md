A React Native e-commerce template with visual search functionality built using Expo.

**Features:**
- User authentication system
- Product catalog and search
- Shopping cart management
- Visual search using TensorFlow
- Order processing and history

**Technology Stack:**
React Native, Expo, Redux, React Navigation, TensorFlow for visual search

**Project Structure:**
- `/assets` - Fonts and images
- `/components` - Reusable UI components
- `/screens` - Application screens
- `/navigation` - Navigation configuration
- `/store` - State management
- `/utils` - Helper functions

**Setup:**
Requires Node.js, Expo CLI, and specific TensorFlow packages installed with force flag due to React 18 compatibility.
